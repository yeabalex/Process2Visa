# Process2Visa - Travel Consultancy Platform

This is a [Next.js](https://nextjs.org) project for Process2Visa, a comprehensive travel consultancy platform that helps users with visa applications, travel planning, and related services.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Features

- **Service Discovery**: Browse and explore various travel and visa services
- **User Authentication**: Secure login via Telegram integration
- **Service Management**: Purchase and manage travel consultancy services
- **Progress Tracking**: Monitor your visa application and travel planning progress
- **Responsive Design**: Modern UI that works on all devices
